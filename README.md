# Swiftly MVP Website

Swiftly is a front-end MVP website for a peer-to-peer parking marketplace in Honolulu, Hawaii. It is built as a simple static website using HTML, CSS, and JavaScript.

## Project structure

- `index.html` - Home page
- `how-it-works.html` - Marketplace explanation for drivers and hosts
- `locations.html` - Mock parking search page with sample Honolulu areas
- `listings.html` - Sample pricing and host listings
- `host.html` - Become a Host page with a demo sign-up form
- `about.html` - Business concept and e-business model overview
- `contact.html` - Contact form and FAQ page
- `styles.css` - Shared styling, layout, and responsive design
- `app.js` - Mock listings data and simple UI behavior

## How to run the project

1. Open the project folder in your code editor.
2. Open `index.html` in any web browser.
3. To browse the site, use the navigation links at the top of each page.

If you want a local development server, you can also use one of these options:

- VS Code Live Server extension
- `python -m http.server`
- Any simple static server tool

## How to edit the project

- Update text content directly inside each HTML file.
- Change colors, spacing, layout, and responsive styles in `styles.css`.
- Edit mock parking listings or interactive behavior in `app.js`.

## Notes

- This project uses mock data only.
- No backend or database is included yet.
- The structure is ready to be extended later with real booking logic, authentication, and database connections.
